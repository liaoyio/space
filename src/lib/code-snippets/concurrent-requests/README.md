# 如何解决页面请求接口大规模并发问题

先交代背景： 其实这种批量大规模发送请求的场景在大一点的项目挺常见的，比如数据平台、实时监控平台、从服务器批量获取数据。
这个时候如果我们不对对请求做对应的阈值或流量进行控制，就很容易出现故障。

回到前端，我们如果想要解决这个大规模并发呢？
